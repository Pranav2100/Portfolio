Overview

Run. Dodge. Survive. Repeat.

Endless Rush is a fast-paced 3D endless runner built in Unity — inspired by games like Subway Surfers. Players sprint through procedurally generated environments, dodging obstacles, collecting coins, and chasing high scores as the speed relentlessly climbs.

The game features lane-switching mechanics, obstacle variety, power-ups, and an escalating difficulty curve that keeps every run feeling fresh and challenging. Clean controls and satisfying feedback make it easy to pick up and impossible to put down.

Development Process

Built in Unity using C# — the game uses an object pooling system for performance, procedural track generation for infinite variety, and a modular obstacle system that allows new challenges to be added quickly.

Key Features & Systems

🏛

Level Design

Hand-crafted temple environments with layered traps, secret passages, and progression pacing designed to keep players engaged.

🧩

Puzzle Mechanics

Interactive puzzles using Blueprint scripting — pressure plates, movable objects, and environmental triggers woven into the level layout.

💡

Lighting & Atmosphere

UE5 Lumen-powered dynamic lighting with torches, god rays, and shadow play to create a moody, cinematic atmosphere throughout.

🎨

3D Asset Integration

Sourced and integrated high-quality 3D assets — optimised for performance while maintaining visual fidelity across the entire game world.

🔊

Sound Design

Ambient soundscapes, triggered audio cues, and reactive sound effects that reinforce exploration and add tension to trap sequences.

⚙️

Blueprint Scripting

All game logic built using UE5 Blueprint — from character movement and camera control to interactive objects and game state management.
