How I Work

A Process Built
Around Players

01

Concept

Define the core player fantasy, emotional arc, and the single mechanic that makes it unique.

02

Prototype

Gray-box in 48 hours. If the loop isn't fun without art, it won't be fun with it.

03

Architecture

Scalable, event-driven systems that let the game grow without technical debt.

04

Polish

Screen shake, SFX, animation curves. The last 20% of dev adds 80% of the feel.

05

Ship

Structured playtests, drop-off analysis, iteration — then release with confidence.
